# ROSA NOIRE — Luxury Cosmetics

Люксовая косметика Rosa Noire — искусство безупречности, созданное для тех, кто ценит роскошь в деталях.

## 🌟 Особенности

- ✨ Элегантный дизайн с использованием цветовой схемы золото/чёрный
- 📱 Полная адаптивность для мобильных устройств (480px, 768px, 1024px+)
- 🛒 Функциональная корзина покупок с сохранением данных
- ❤️ Список желаний (wishlist)
- 👤 Система авторизации пользователей
- 🔍 Фильтрация товаров по категориям
- 📱 Отзывчивое меню-бургер для мобильных устройств

## 📂 Структура проекта

```
Сайт/
├── index.html              # Главный HTML файл
├── styles/
│   └── main.css            # Основные стили и адаптивность
└── js/
    ├── auth.js             # Логика авторизации
    ├── auth-ui.js          # UI авторизации
    ├── cart.js             # Логика корзины
    ├── products.js         # Данные товаров
    ├── ui.js               # Управление пользовательским интерфейсом
    ├── utils.js            # Вспомогательные функции
    └── wishlist.js         # Логика списка желаний
```

## 🎨 Используемые технологии

- HTML5
- CSS3 (в т.ч. Grid, Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Cormorant Garamond + Montserrat)

## 🚀 Быстрый старт

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/rosa-noire.git
```

2. Откройте файл `Сайт/index.html` в браузере

3. Наслаждайтесь роскошью 💎

## 💻 Функциональность

### Авторизация
- Регистрация новых пользователей
- Вход в аккаунт
- Сохранение данных пользователя в localStorage

### Каталог товаров
- 8 товаров в 3 категориях (для губ, лица, глаз)
- Фильтрация по категориям
- Детальный просмотр информации о товаре

### Корзина покупок
- Добавление/удаление товаров
- Изменение количества товаров
- Прокрутка товаров в меню
- Сохранение в localStorage

### Список желаний
- Добавление товаров в список желаний
- Управление списком
- Счётчик товаров
