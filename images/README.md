# Медиа-файлы сайта Rosa Noire

Папка для всех изображений и медиа-файлов сайта.

## Структура:

- `products/` - Фотографии товаров для каталога
  - 8 изображений товаров по категориям
  - Размер: 400x480px минимум

## Как добавлять изображения:

1. **Для товаров**: поместите JPG/PNG в папку `products/`
2. **Фотографии должны быть**: роскошные, высокого качества, соответствующие стилю бренда
3. **Имена файлов**: используйте латинские буквы и нижний регистр (например: `lipstick.jpg`)

## Общие рекомендации:

- Все изображения в формате JPG предпочтительнее (для оптимизации)
- Размер файлов: по 50-200 KB каждое
- Размер изображений: 400x480px для карточек товаров
- Стиль: минималистичный, элегантный, с фокусом на товар
