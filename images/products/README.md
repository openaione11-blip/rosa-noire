# Фотографии товаров

В этой папке хранятся изображения товаров для каталога сайта.

## Требуемые изображения:

### Губы (lips)
- `lipstick.jpg` - Велюровая помада ROUGE NOIRE
- `tint.jpg` - Жидкий тинт ROSE KISS
- `gloss.jpg` - Блеск-бальзам SHIMMER LUXE

### Лицо (face)
- `concealer.jpg` - Консилер RADIANCE
- `powder.jpg` - Прессованная пудра VELVET MATTE
- `foundation.jpg` - Тональная основа PERFECT SKIN

### Глаза (eyes)
- `mascara.jpg` - Тушь VOLUME LUXURY
- `eyeshadow.jpg` - Палетка теней NOIRE PALETTE

## Рекомендации для фотографий:

✨ **Размер**: минимум 400x480px (рекомендуется 600x720px)
✨ **Формат**: JPG, PNG (JPG для оптимизации размера файла)
✨ **Качество**: сохранять в формате JPG с качеством 85-90%
✨ **Стиль**: роскошные фотографии товаров на боковом фоне, в согласовании с цветовой схемой бренда (чёрный, золотой)

## Как добавлять фотографии:

1. Поместите готовые фотографии в эту папку (`images/products/`)
2. Назовите их в соответствии с именами файлов выше
3. Сайт автоматически будет использовать нужное изображение для каждого товара

Если изображение не загружается, проверьте:
- Правильное имя файла
- Путь соответствует: `images/products/filename.jpg`
- Файл находится именно в папке `images/products/`
